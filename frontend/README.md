# Frontend - Google Intelligent Group

Next.js frontend application for the Multi-Agent System.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Set environment variables (optional):
```bash
# Create .env.local file
NEXT_PUBLIC_API_URL=http://localhost:8000
```

3. Run development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Modern React-based UI with Next.js 14
- Real-time query input and response display
- Agent dashboard showing outputs from:
  - Supervisor Agent
  - GoogleMap Agent
  - Calendar Agent
  - Telephone Agent
  - Research Agent
- Responsive design

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Axios for API calls
